## 项目介绍

此项目为vue3+ts的练手项目，难度并不高。主要是将vue2做过的案例用vue3和ts再写一遍

### 计数器（实现最基本的功能，数字加减）

### 图片切换（类似于简化版的轮播图）

### 天气查询 （使用axios发送请求 查询天气）
